## Overview
This repository contains a **sales analytics case study** built with **Power BI** using a synthetic dataset.
The project focuses on data modeling, advanced DAX, and interactive reporting techniques commonly used in real-world business dashboards.

The dataset is fictional and created solely for learning and portfolio demonstration purposes.

---

## Key Features
- Star schema data modeling (Fact & Dimension tables)
- Advanced DAX measures and Time Intelligence:
  - YTD, MTD
  - Month-over-Month (MoM%)
  - Year-over-Year (YoY%)
  - Running Total
  - Rolling 3-Month Average
- Interactive dashboards with:
  - KPI cards
  - Tooltips
  - Drillthrough pages
  - Bookmarks
- What-if scenario analysis
- Role-based view (basic RLS concept)
- Clean and presentation-ready report pages
